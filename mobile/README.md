# Song Generator Pro - Mobile App

AI-powered music generation on iOS and Android using multi-model ensemble approach.

## 🎵 Features

### Core Functionality
- **AI Song Generation**: Create professional music from text descriptions
- **Voice Input**: Describe your song using voice-to-text
- **Multi-Model Ensemble**: Uses 3+ AI models to generate the best quality
- **Real-time Progress**: Live updates during generation
- **Offline Library**: Songs stored locally for offline playback
- **Professional Audio Player**: Full-featured player with waveform visualization
- **Comparison Mode**: Swipe through multiple AI generations and pick the best

### Generation Options
- **8 Genres**: Pop, Rock, Electronic, Jazz, Classical, Hip Hop, R&B, Country
- **8 Moods**: Happy, Sad, Energetic, Calm, Romantic, Dark, Uplifting, Mysterious
- **Flexible Duration**: 30s to 5 minutes
- **4 Strategies**:
  - Fastest (~30s, $0.03)
  - Best Quality (~45s, $0.05)
  - Ensemble Top 3 (~2-3min, $0.15) [Default]
  - All Models (~3-5min, $0.20+)

### User Experience
- **Dark Theme**: Beautiful, modern dark UI
- **Gesture Controls**: Swipe navigation in comparison mode
- **Auto Mastering**: Optional professional mastering
- **Quality Metrics**: See spectral clarity, dynamic range, and more
- **Library Management**: Filter, sort, and organize your songs
- **Persistent Storage**: MMKV for fast local storage

## 🚀 Quick Start

### Prerequisites

```bash
# Install Node.js 18+ and npm
node -v  # Should be v18 or higher
npm -v

# Install Expo CLI globally
npm install -g expo-cli

# Install EAS CLI for building
npm install -g eas-cli
```

### Installation

```bash
# Navigate to mobile directory
cd mobile

# Install dependencies
npm install

# Start development server
npm start
```

### Running on Devices

**iOS Simulator** (Mac only):
```bash
npm run ios
```

**Android Emulator**:
```bash
npm run android
```

**Physical Device**:
1. Install Expo Go app from App Store or Play Store
2. Scan QR code from `npm start`

## 🔧 Configuration

### Connect to Backend

Update the API URL in `services/api.ts`:

```typescript
const API_BASE_URL = __DEV__
  ? 'http://localhost:3000/api'              // Local development
  : 'https://your-app.vercel.app/api';       // Production
```

**Important**: For iOS simulator, use `localhost`. For Android emulator, use `10.0.2.2` instead of `localhost`.

### Environment Variables

Create `.env` file (optional):

```env
EXPO_PUBLIC_API_URL=https://your-app.vercel.app/api
```

## 📱 Building for Production

### iOS App Store

```bash
# Login to Expo
eas login

# Configure iOS build
eas build:configure

# Create production build
eas build --platform ios --profile production

# Submit to App Store
eas submit --platform ios
```

**Requirements**:
- Apple Developer Account ($99/year)
- Bundle ID configured in `app.json`
- App icons and splash screens

### Android Play Store

```bash
# Create production build
eas build --platform android --profile production

# Submit to Play Store
eas submit --platform android
```

**Requirements**:
- Google Play Developer Account ($25 one-time)
- Package name configured in `app.json`
- Signing key (automatically generated by EAS)

### Local Builds

**iOS** (Mac only):
```bash
eas build --platform ios --profile development --local
```

**Android**:
```bash
eas build --platform android --profile development --local
```

## 🎨 Customization

### App Name & Icons

1. **App Name**: Edit in `app.json`:
```json
{
  "expo": {
    "name": "Your App Name",
    "slug": "your-app-slug"
  }
}
```

2. **App Icon**: Replace `assets/icon.png` (1024x1024)

3. **Splash Screen**: Replace `assets/splash.png`

### Theme Colors

Edit colors in component styles:
```typescript
// Primary: #8b5cf6 (purple)
// Background: #0a0a0a (black)
// Card: #1a1a1a (dark gray)
// Border: #333 (gray)
```

### Generation Settings

Update defaults in `store/useStore.ts`:
```typescript
settings: {
  autoMastering: true,
  defaultStrategy: 'ensemble-top3',
  audioQuality: 'high',
  notifications: true,
  theme: 'dark',
}
```

## 🐛 Troubleshooting

### Metro Bundler Issues
```bash
# Clear cache
npx expo start --clear

# Reset cache
rm -rf node_modules
npm install
npx expo start --clear
```

### iOS Build Issues
```bash
# Clear Xcode derived data
rm -rf ~/Library/Developer/Xcode/DerivedData

# Reinstall pods
cd ios && pod install && cd ..
```

### Android Build Issues
```bash
# Clean gradle
cd android && ./gradlew clean && cd ..

# Reset gradle cache
rm -rf ~/.gradle/caches/
```

### Audio Playback Issues
- **iOS**: Ensure `playsInSilentModeIOS: true` in audio config
- **Android**: Check audio focus settings
- **Both**: Verify audio URLs are HTTPS
